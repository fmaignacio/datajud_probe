# 00 — Contexto da Pesquisa

## Tema
Investigar como LLMs reconfiguram produção, validação e uso de conhecimento institucional no setor público brasileiro.

## Pergunta teórica
Como o uso de LLMs em contextos institucionais reconfigura a produção, a validação e o uso do conhecimento institucional no setor público brasileiro?

## Pergunta empírica provisória
Como saídas geradas por LLMs diferem de decisões humanas institucionais em casos jurídicos estruturados com dados públicos?

## Escopo atual (fase)
- Fase: exploração e validação de base de dados.
- Prioridade: avaliar qualidade/cobertura dos dados antes de qualquer comparação LLM vs decisão institucional.
- Corpus semântico principal: STJ Íntegras.
- Base estrutural/processual: DataJud.

## Hipóteses provisórias
- LLMs podem operar como forma de delegação cognitiva implícita.
- Divergências LLM vs decisão institucional tendem a ser não aleatórias.

## Delimitações
- Não é estudo de substituição decisória humana.
- Decisão institucional não é tratada como verdade absoluta.
- Conclusões causais fortes só após validação robusta.
